Structura repetitivă
Se foloseşte atunci când se impune repetărea unei secvenţe de instrucţiuni. Apare în următoarele variante:

  1. Structura repetitivă cu număr nedeterminat(necunoscut) de paşi
    1. Structura repetitivă cu test intiţial
    2. Structura repetitivă cu test final
  2. Structura repetitivă cu număr determinat de paşi (cu contor)

Structura repetitivă cu test intiţial (prefixată) se numeşte astfel deoarece la intrarea în structură se testează mai întâi valoarea de adevăr a unei condiţii, iar în funcţie de rezultat se execută în mod repetăt o secvenţa de instrucţiuni, sau se părăseşte structura.

Reprezentarea în pseudocod:

┌cât timp C execută
│   bloc_instrucţiuni
└∎

Mod de execuţie
  1. Se evaluează condiţia C
  2. Dacă valoarea de adevăr a condiţiei C este 1/True, se execută blocul de instrucţiuni după care se revine la testarea condiţiei C
  3. Când valoarea de adevăr a condiţiei C devine 0/False, se părăseşte structura.
Observaţie: Dacă la prima evaluare a condiţiei C, valoarea ei este 0/False, atunci blocul de instrucţiuni nu va fi executat nici macăr o dată.

Exemplu:
Se citeşte un număr natural n. Se cere să se determine şi să se afişeze numărul cifrelor sale. (pentru n=1342, se va afişa 4).

citeşte n
nr←0
┌cât timp n≠0 execută
│   n←[n/10]
│   nr←nr+1
└∎
scrie nr

Structura repetitivă cu test final (postfixată) presupune executarea unui bloc de instrucţiuni şi apoi testarea valorii de adevăr a unei condiţii, în funcţie de care se părăseste sau nu structura. Se recomandă utilizarea ei atunci când există certitudinea executării cel putin o dată a blocului de instrucţiuni inclus în structură.

Reprezentarea în pseudocod:

  1. ┌execută
    │   bloc_instrucţiuni
    │cât timp C
    └∎

    Mod de execuţie
    1. Se execută necondiţionat blocul de instrucţiuni.
    2. Se evaluează condiţia C.
    3. Daca valoarea de adevăr a condiţiei este 1/True, atunci se va relua execuţia blocului de instrucţiuni.
    4. Atunci când valoarea de adevăr a condiţiei va fi 0/False, se va părăsi structura repetitivă.

    2. Exemplu:
    Se citesc de la tastatură numere intregi până la citirea unui 0. Ştiind că primul număr citit este nenul, să se afişeze numărul de numere citite, fără 0.

    nr←0
    ok←1
    ┌execută
    │   citeşte x
    │   daca x!=0 atunci
    │      nr←nr+1
    │   altfel
    │      ok=0
    │cât timp ok≠0
    └∎
    scrie nr

  2. ┌repetă
    │   bloc_instrucţiuni
    │până când C
    └∎

    Mod de execuţie
    1. Se execută necondiţionat blocul de instrucţiuni.
    2. Se evaluează condiţia C.
    3. Daca valoarea de adevăr a condiţiei este 0/False, atunci se va relua execuţia blocului de instrucţiuni.
    4. Atunci când valoarea de adevăr a condiţiei va fi 1/True, se va părăsi structura repetitivă.

Exemplu:
Se citeşte un şir de număr natural până la citirea unui număr de exact două cifre. Să se afiseze predecesorul şi succesorul acestuia.

citeşte n
┌repetă
│   citeşte n
│până când n≥10 şi n≤99
└∎
scrie n-1
scrie n+1

Structura repetitivă cu număr determinat de paşi (cu contor) se utilizează atunci când se cunoa&scecil;te, înaintea executării blocului de instrucţiuni de câte ori trebuie să se execute acesta.

Reprezentarea în pseudocod:

┌pentru i←vi, vf, pas execută
│   bloc_instrucţiuni
└∎

Observaţii:
  1. i este numit contor
  2. vi reprezintă valoarea intiţială a contorului
  3. vf reprezintă valoarea finală a contorului
  4. pas reprezintă valoarea cu care variază contorul
  5. dacă pas este un număr pozitiv, i va creşte de la vi la vf
  6. dacă pas este un număr negativ, i va scădea de la vi la vf

Mod de execuţie (pentru pas>0)
  1. Se atribuie contorului valoarea intiţială
  2. Se compară valoarea contorului cu valoarea finală. Daca i este mai mic sau egal cu vf, se execută blocul de instrucţiuni, după care contorul se incrementează cu pas (i←i+pas).
  3. Atunci când valoarea contorului devine mai mare decât vf se va părăsi structura repetitivă.
Observaţie: Daca pasul este 1, de obicei, nu se mai scrie.

Exemplu:
Se citeşte un număr natural nenul n. În continuare, se citeşte un şir de n numere naturale. Să se determine şi să se afişeze câte dintre acestea sunt pare.
pentru n=7 şi sirul 25 2 62 8 7 12 361 se va afişa 4

citeşte n
nr←0
┌pentru i←1, n execută
│   citeşte x
│   daca x&2=0 atunci
│      nr←nr+1
└∎
scrie nr
