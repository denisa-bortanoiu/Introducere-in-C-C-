Structura alternativă
Se foloseşte pentru a alege o alternativă din două posibile.

Reprezentare în pseudocod:

┌dacă expresie_logică atunci
│   bloc_instructiuni1
│altfel
│   bloc_instructiuni2
└∎

Mod de execuţie:
În cazul în care valoarea de adevăr a expresiei logice este 1/True, se va executa bloc_instructiuni1, iar în caz contrar se va executa bloc_instructiuni2.

Observaţii:

  1. bloc_instructiuni1 şi bloc_instructiuni2 pot conţine orice fel de structură, deci şi o alta structuă alternativă.
  2. ramura altfel poate lipsi.
  3. intotdeauna se va executa un singur bloc de instrucţiuni în funcţie de valoarea de adevăr a expresiei logice, după care se va părăsi structura alternativă.

Exemple:

  1. Se citeşte o valoare pentru o variabilă de tip întreg. Se cere să se afişeze valoarea absolută a numărului citit.

    citeşte a
    ┌dacă a≥0 atunci
    │   scrie a
    │altfel
    │   scrie -a
    └∎

  2. Se citesc 3 valori. Să se afişeze cea mai mare valoare.

    citeşte a, b, c
    ┌dacă a≥b atunci
    │   ┌dacă a≥c atunci
    │   │   scrie a
    │   └∎
    │altfel
    │   ┌dacă b≥c atunci
    │   │   scrie b
    │   │altfel
    │   │   scrie c
    │   └∎
    └∎
