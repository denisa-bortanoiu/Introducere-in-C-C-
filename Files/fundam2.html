  • Operarea pe cifre
  • Divizorii unui numar
  • C.m.m.d.c.
  • Exerciţii
Cel mai mare divizor comun al numerelor a şi b(c.m.m.d.c.(a, b)) poate fi determinat folosind algoritmul lui Euclid:
  1. Se citesc numerele a şi b.
  2. Se calculează restul împărţirii lui a la b.
  3. Cât timp restul este nenul, a ia valoarea lui b, b valoarea restului şi se calculează din nou restul împărţirii lui a la b.
  4. La finalul repetiţiei, ultimul rest nenul (b) este c.m.m.d.c(a, b).
Se citessc două numere a, b, de maxim patru cifre. Se cere să se determine şi să se afiseze c.m.m.d.c.(a,b).
Exemplu: pentru n=1342, se va afişa 4.

#include<iostream.h>
int main () {
   int a, b, r;
   cin>>a>>b;
   r=a%b;
   while (r!=0) {
      a=b;
      b=r;
      r=a%b; }
   cout<<b;
return 0; }

Observaţii!
  1. două numere se numesc prime între ele dacă c.m.m.d.c.(a, b)=1.
  2. c.m.m.d.c. pe un şir de numere a1, a2, a3...an este calculat ca
          c.m.m.d.c.(c.m.m.d.c(ak-2, ak-1), ak) unde k∈{3, 4, ..., n}.
  3. cel mai mare multiplu comun a două numere a şi b este dat de formula
          (a*b)/c.m.m.d.c.(a, b)
    În implementare, se salvează valorile iniţiale ale variabilelor a şi b în variabile auxiliare! (procesul de determinare a c.m.m.d.c(a,b) schimbă valorile celor două variabile)
