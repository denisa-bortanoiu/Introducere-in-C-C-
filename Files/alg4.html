I.Exerciţii rezolvate
Alege răspunsul corect:

  1. Ce valoare rezultă în urma executării expresiei [3*5/2]*2?

    1. 3
    2. 15
    3. 14
    4. 7

  2. Care din următoarele afirmaţii este falsă?

    1. < este operator relaţional
    2. % este operator aritmetic
    3. not este operator logic
    4. ≠ este operator logic

  3. Ştiind că a şi b sunt două variabile reale, ce tip are valoarea rezultată în urmă evaluării expresiei (a>3) and (b=5)?

    1. întreg
    2. real
    3. logic
    4. caracter

  4. Ce valoare va avea variabila x la sfârşitul executării secvenţei următoare daca x are valoarea iniţială 10?

    x←2*x/5
    x←x+10

    1. 25
    2. 40
    3. 4
    4. 14

  5. Şiind că n este o variabilă ce reţine un număr natural de 3 cifre, care dintre următoarele expresii determină cifra sutelor numărului memorat în n?

    1. n%10
    2. n%100
    3. [n/10]
    4. [n/100]

  6. Şiind că n este o variabilă ce reţine un număr natural de 3 cifre, care dintre următoarele expresii determină cifra zecilor numărului memorat în n?

    1. [n%100/10]
    2. n%100
    3. [n/10%100]
    4. [n/10]

  7. Care dintre următoarele expresii are valoarea 1/True dacă şi numai dacă numărul natural c este un multiplu comun al numerelor naturale nenule a şi b?

    1. (c%a=0) sau (c%b=0)
    2. (c/a=0) sau (c/b=0)
    3. (c%a=0) şi (c%b=0)
    4. (c/a=0) şi (c/b=0)

  8. Care au fost valorile iniţiale ale variabilelor întregi x şi y la începutul executării secvenţei pseudocod de mai jos dacă la finalul executării x are valoarea 2007, iar y are valoarea 2009?

    x←x-1
    y←2*x+y
    x←2*x+1

    1. x=1002 şi y=5
    2. x=1004 şi y=3
    3. x=3 şi y=1004
    4. x=2007 şi y=2009

Răspunsuri: 1. c; 2. d; 3. c; 4. d; 5. d; 6. a; 7. c; 8. b

II. Exerciţii propuse pentru rezolvare
Scrieţi o expresie logică care are valoarea 1 dacă şi numai dacă:
  1. valoarea memorată în variabila x este în intervalul (a, b)
  2. valoarea memorata în variabila x este în intervalul [a, b]∪(c, d)
  3. numarul natural memorat în variabila n este par şi are exact 2 cifre
  4. valorile variabilelor a, b şi c sunt în ordine strict crescătoare

Evaluaţi expresiile:
  1. n%10+n/10%10+n/100%10, unde n este o variabilă întreagă cu valoarea 32735
  2. a≤b şi b≤c sau c%2=1, unde a, b şi c sunt 3 variabile întregi cu valorile 5, 7, respectiv 8
  3. 1+(2*a/b+a+2*b/3*c)/2+c, unde a, b si c sunt 3 variabile întregi cu valorile 4, 6, respectiv 2
