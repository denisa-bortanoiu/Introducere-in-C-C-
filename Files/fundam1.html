Structura generală
Algoritmul folosit este acela de acces la cifrele unui număr. Paşii sunt, după cum urmează:
  1. Se citeşte numărul n ce urmează a fi prelucrat.
  2. Se împarte numărul la 10, cu rest
  3. Rest al împărţirii, ultima cifră devine disponibilă pentru afişare/operare
  4. Numărul devine n←[n/10]
  5. Se revine repetat la pasul 2, dacă n!=0 sau se opreşte repetiţia în caz contrar.
Determinarea cifrei maxime dintr-un număr
Se citeşte un număr natural nenul n de maxim 9 cifre. Se cere să se determine şi să se afişeze cea mai mare cifră a numărului n.
Exemplu: pentru n=1342, se va afişa 4.

#include<iostream.h>
int main () {
   long n;
   int max;
   cin>>n;
   max=n%10;
   while (n!=0) {
      if (max>n%10)
      max=n%10;
      n=n/10; }
   cout<<max;
return 0; }

Observaţie! Pentru minim, se înlocuieşte condiţia max>n%10 cu min<n%10

Determinarea sumei/produsului cifrelor unui număr

#include<iostream.h>
int main () {

return 0; }

Oglinditul unui număr. Număr palindrom

Oglinditul unui număr este numărul citit de la dreapta la stânga.


Se citeşte un număr natural nenul n de maxim 9 cifre. Se cere să se determine şi să se afişe oglinditul său.
Exemplu: pentru n=1342, se va afişa 2341.

#include<iostream.h>
int main () {
   long n, ogl;
   cin>>n;
   ogl=0;
   while (n!=0) {
      ogl=ogl*10+n%10;
      n=n/10; }
   cout<<ogl;
return 0; }

Un număr este palindrom dacă citit de la dreapta la stânga este egal cu numărul citit de la stânga la dreapta (exemple: 12321, 232, 4, 24342). În alte cuvinte, dacă un număr este egal cu oglinditul său, acesta este palindrom.


Se citeşte un număr natural nenul n de maxim 9 cifre. Se cere să se scrie pe ecran "da", dacă numărul este palindrom şi "nu" în caz contrar.
Exemplu: pentru n=1342, se va afişa "nu".

#include<iostream.h>
int main () {
   long n, ogl, aux;
   cin>>n;
   ogl=0;
   aux=n;
   while (n!=0) {
      ogl=ogl*10+n%10;
      n=n/10; }
   if (aux==ogl) cout<<"da";
   else cout<<"nu";
return 0; }


Construcţii de numere
Se citeşte un număr natural nenul n de maxim 9 cifre. Se cere să construiască în variabila x numărul alcătuit din cifrele sale impare, privite de la dreapta la stânga. Dacă numărul nu are cifre impare, se va afişa un mesaj corespunzător.
Exemplu: pentru n=1342, se va afişa 31.


#include<iostream.h>
int main () {
   long n, x=0;
   cin>>n;
   do {
      if (n%2==1) x=x*10+n%10;
      n=n/10; }
   while (n!=0);
   if (x!=0) cout<<x;
   else cout<<"nu exista cifre impare";
   //daca x ramane 0, atunci nu au existat cifre impare
return 0; }

Se citeşte un număr natural nenul n de maxim 9 cifre. Se cere să construiască în variabila x numărul alcătuit din cifrele sale pare, privite de la stânga la dreapta. Daca numărul nu are cifre pare, se va afişa un mesaj corespunzător.
Exemplu: pentru n=1342, se va afişa 42.

#include<iostream.h>
int main () {
   long n, x=0, p=1;
   cin>>n;
   do {
      if (n%2==0) {
         x=x+n%10*p;
         p=p*10; }
      n=n/10; }
   while (n!=0);
   if (p!=1) cout<<x;
   else cout<<"nu exista cifre pare";
   //exista posibilitatea ca singura cifra pară a numărului citit sa fie 0, caz în care x are valoarea 0, însa p se schimba
return 0; }
