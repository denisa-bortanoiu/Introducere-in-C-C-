Un program este alcătuit din una sau mai multe funcţii, funcţia main fiind prima care se execută şi ea nu poate lipsi. Celelalte funcţii sunt definite de utilizator.
Orice funcţie este formată din:
  1. antet : tip nume_funcţie ([listă parametri formali])
    • O funcţie poate să returneze o valoare de un anumit tip(int, long, double, etc) sau să nu returneze nicio valoare, caz în care este de tip void.
    • Parametri formali sunt identificatori de variabile pe care urmează să le foloseasca funcţia atunci când se va executa. Pentru a comanda execuţia unei funcţii, aceasta trebuie apelată. La apel, parametrii formali sunt înlocuiţi cu valori concrete, numite parametri actuali.
  2. corp : { declaraţii; instrucţiuni; }
    • În cazul în care tipul unei funcţii este diferit de void, corpul ei trebuie să conţina cel puţin o instrucţiune de tip return valoare/expresie.
Funcţia main trebuie să fie de tip int. Ea va returna valoarea 0 la sfârşitul setului de instrucţiuni. Nu are parametri.

În C/C++ există şi funcţii predefinte, care işi au definiţia(prototipul) în fişiere antet/ headere/ biblioteci. predefinita, la inceputul programului trebuie scrisa o directiva preprocesor astfel: #include<nume_header>.

Des folosite, sunt functiile următoare, ale căror prototipuri se găsesc în fişierul antet math.h:
Tipul argumentuluiSemnificaţieExemplu
pow(x,y)  x şy de tip double  xy  a=pow(2, 3); //a=8
abs(x)  x de tip double  valoarea absolută a argumentului  a=abs(-2); //a=2
sqrt(x)  x de tip double  radical din x (x≥0)  a=sqrt(25); //a=5
sin(x)  x de tip double  sinusul lui x  a=sin(0); //a=0
cos(x)  x de tip double  cosinul lui x  a=cos(0); //a=1
atan(x)  x de tip double  unghiul a carui tangentă este x  a=atan(0); //a=0
log(x)  x de tip double  logaritm natural din r (r>0)  a=log(1); //a=0
ceil(x)  x de tip float  cel mai mic întreg ≥r  a=ceil(12.23); //a=13
floor(x)  x de tip float  cel mai mare întreg ≤r  a=floor(1.34); //a=1

Comentariile într-un program sunt încadrate între /* ş*/ sau între // şi sfârşitul liniei.
Variabilele care nu sunt declarate în cadrul unei funcţii se numesc variabile globale. Ele vor fi iniţializate cu valoarea 0. Variabilele declarate în corpul unei funcţii vor fi iniţializate cu valori aleatoare.

Exemplu:
Să se scrie un program C/C++ care calculează cel mai mic divizor propriu al numărului natural nenul x citit de la tastatură.

#include<iostream.h>
#include<math.h>
int main ()
{
   int x, d, r;
   cin>>x;
   r=sqrt(x);
   for(d=2; d<=r; d++)
      if (x%d==0)
      {
         cout<<d;
         break;
      }
   return 0;
}
