Divizori proprii şi improprii
Fie x un număr natural nenul.
Dacă x%a=0(unde a este un număr natural nenul), atunci a se numeşte divizor al lui x.
Dacă x=1, atunci el are un singur divizor (pe el însuşi).
Dacă x>1, atunci el are cu siguranţa cel puţin 2 divizori: 1 şi x. Acestia se numesc divizori improprii.
Orice alti divizori ai lui x se numesc divizori proprii. Dacă d este divizor propriu al lui x, atunci
      d∈{2, 3, ..., x/2}.

Aplicaţie
Să se afişeze toţi divizorii proprii ai numărului natural nenul n citit de la tastatură (n≤1000).
Exemplu: pentru n=24 se va afişa 2 3 4 6 8 12

#include<iostream.h>
int main () {
   int n, d, nr;
   cin>>n;
   nr=0;
   //nr are semnificaţia unui contor, numără câţi divizori proprii are numărul n. Dacă nr ramane 0 atunci n nu are divizori proprii
   for (d=2; d<=n/2; d++)
      if (x%d==0) {
         cout<<d<<' ';
         nr++; }
   if (nr==0) cout<<"numărul nu are divizori proprii";
return 0; }


Cel mai mic divizor propriu
Cel mai mic divizor propriu(dmin) al lui x aparţine mulţimii {2, 3, ... √x}. Fiind cel mai mic divizor propriu, acesta este şi prim.
Cel mai mare divizor propriu al lui x este dat de formula: x/dmin.

Aplicaţie
Să se afişeze cel mai mic şi cel mai mare divizor propriu al numărului natural nenul n citit de la tastatură (n≤1000).
Exemplu: pentru n=21 se va afişa 3

#include<iostream.h>
#include<math.h>
int main () {
   int n, d, r, ok;
   cin>>n;
   r=sqrt(n);
   ok=0; //Dacă există divizor propriu, ok=1; altfel, ok=0;
   for (d=2; d<=r; d++)
      if (x%d==0) {
         cout<<d<<' ';
         break;
         ok=1;}
   if (ok==1) cout<<x/d;
   else cout<<"numărul nu are divizori proprii";
return 0; }


Verificarea primalităţii unui număr
Un număr este prim dacă nu are divizori proprii.
În implementare, cautăm un divizor propriu (dmin∈{2, 3, ... √x}). La găsirea acestuia, oprim repetiţia, pentru că numărul nu este prim şi semnalizăm acest lucru printr-o variabilă.

Aplicaţie
Să se verifice dacă numărul natural nenul n citit de la tastatură (n≤1000) este prim.
Exemplu: pentru n=21 se va afişa "nu". pentru n=29 se va afisa "da".

#include<iostream.h>
#include<math.h>
int main () {
   int n, d, r, ok;
   cin>>n;
   r=sqrt(n);
   ok=1; //Dacă n este prim, ok=1; altfel, ok=0; pornim de la ipoteza ca n este prim
   for (d=2; d<=r; d++)
      if (x%d==0) {
         ok=0;
         break;}
   if (ok==1) cout<<"da";
   else cout<<"nu";
return 0; }


Descompunerea unui număr în factori primi
Orice număr natural nenul şi diferit de 1 se poate scrie ca descompunere de factori primi:
    n=f1 p1 ∙ f2 p2 ∙...∙ fn pn
unde f1, f2, ...fn sunt divizorii primi ai numărului n,
iar p1, p2, ...pn sunt puterile la care aceştia se regăsesc în descompunerea numărului.
Numărul de divizori naturali ai unui număr este
    (1+p1)∙(1+p2)∙(1+p3)∙...∙(1+pn)

Aplicaţie
Să se descompună în factori primi un numărul natural nenul n citit de la tastatură (n≤1000). Se vor afişa factorii primi, alături de puteri.
Exemplu: pentru n=24 se va afisa 2^3   3^1. (23 ∙ 3=24)

#include<iostream.h>
int main () {
   int n, d, p;
   cin>>n;
   d=2; //posibil divizor al lui n
   while (n>1) {
      p=0; //p este puterea cu care apare d în descompunerea lui n
      while(n%d==0) {
         p++;
         n=n/d; }
      if (p>0) cout<<d<<'^'<<p<<' ';
      d++; }
return 0; }
